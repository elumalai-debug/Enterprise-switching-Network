# Enterprise Switching Network using Cisco Packet Tracer

## 📌 Project Overview

This project demonstrates the implementation of an enterprise-level switched network using Cisco Packet Tracer. The network is designed with multiple VLANs, Inter-VLAN Routing, EtherChannel, DHCP, DHCP Relay, DNS, and HTTP (IIS) services to simulate a real-world enterprise environment.

The objective of this project is to provide secure network segmentation, high availability, centralized IP address management, and web service accessibility across different departments.

---

## 🚀 Features

- VLAN Configuration
- Inter-VLAN Routing
- DHCP Server
- DHCP Relay Agent

- EtherChannel using PAgP
- Layer 2 Switching
- Layer 3 Switching
